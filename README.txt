============Installation============

1-Click on "Clone or download" and select the option "Open in Visual Studio"
2-In Visual Studio Team Explorer - connect Menu choose your location(your visual studio workspace, otherwise you'll have problem with library references) to clone de the project
3-After finish, the soluction explorer will be open, click os "Solutions and Folders" and select the option "BlissRecApp.sln"
4-Clean the solution and rebuild it
5-Run the project

==============NOTES==============

You may have some problems with library references.
If you cannot build the solution because of missing references, just add to the project the library in "Library" folder in GitHub.

"Library" folder should not be imported to visual studio soluction.
